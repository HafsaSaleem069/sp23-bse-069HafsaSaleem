Project 2: Weather App

The Weather App is a user-friendly web application that provides real-time weather updates and forecasts. By integrating with a weather API, the app allows users to search for any city and instantly view current weather conditions such as temperature, humidity, wind speed, and an icon representing the weather (e.g., sunny, cloudy, rainy). Additionally, the app can display a multi-day forecast, helping users plan ahead. Designed with a clean interface and responsive design, the Weather App ensures a seamless experience across all devices, making it an essential tool for anyone needing accurate weather information.
